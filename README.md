# AI-Powered FAQ Chatbot

This project is an AI-powered FAQ chatbot developed using Python and Flask.
I built this project to understand the basics of chatbot development and how
web applications can be created using Flask.

The chatbot works by responding to predefined frequently asked questions and
provides appropriate answers based on user input.

## Features
- Simple FAQ-based chatbot
- Rule-based response system
- Centered and clean user interface
- Animated background design
- Custom branding

## Technologies Used
- Python
- Flask
- HTML
- CSS

## How to Run the Project
1. Clone or download the repository
2. Create and activate a virtual environment
3. Install required libraries using pip
4. Run the application using `python app.py`
5. Open a browser and visit `http://127.0.0.1:5000`
